# Popular Python Web Frameworks

1. **Django** - A high-level, batteries-included framework that follows the MTV pattern and provides an ORM, admin panel, and authentication out of the box.
2. **Flask** - A lightweight, flexible microframework that gives developers fine-grained control over components and is ideal for small to medium applications.
3. **FastAPI** - A modern, high-performance framework built on Starlette and Pydantic that provides automatic API documentation and native async support.
4. **Tornado** - An asynchronous networking library and web framework designed for handling thousands of simultaneous connections with long-polling and WebSockets.
5. **Pyramid** - A flexible, minimalist framework that scales from single-file applications to large enterprise systems with a focus on configuration flexibility.
