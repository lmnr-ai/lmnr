1. **Django** — A full-stack “batteries-included” framework with ORM, admin, auth, and templating geared toward rapid development of data-driven sites and APIs.
2. **Flask** — A minimal microframework that stays small by default and gains features through extensions, making it flexible for APIs and prototypes.
3. **FastAPI** — A modern framework built on Starlette and Pydantic that emphasizes async support, automatic OpenAPI docs, and type-hint-driven validation.
4. **Starlette** — A lightweight ASGI toolkit for building async web services and routing without prescribing a large application stack.
5. **Tornado** — An async-capable framework and networking library often used for long-lived connections, WebSockets, and high-concurrency I/O-bound apps.
