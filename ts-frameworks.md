1. **Express** — The de facto minimalist HTTP server for Node.js with a huge ecosystem of middleware, widely used for REST APIs and BFF layers.
2. **NestJS** — An opinionated, Angular-inspired framework that uses TypeScript decorators, dependency injection, and modules to structure large server apps.
3. **Fastify** — A performance-focused web framework with a schema-based approach to validation and serialization and a pluggable ecosystem.
4. **Hono** — A small, standards-oriented framework that runs on many runtimes (Node, Bun, Cloudflare Workers) and excels at edge and serverless HTTP.
5. **tRPC** — A type-safe RPC layer over HTTP that lets you call backend procedures from the client with end-to-end TypeScript inference without REST schemas.
